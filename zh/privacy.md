# 隐私政策

**最后更新：2025年2月**

Dash VPN 是一款致力于保护隐私的应用程序。我们承诺**不收集、不存储也不出售**您的任何数据。

## 1. 零日志政策 (Zero-Logs Policy)

我们从根本上尊重您的隐私。Dash VPN **绝对不**收集或存储以下信息：

- **您的网络流量：** 我们不记录您的浏览历史、流量目的地、数据内容或 DNS 查询。
- **您的 IP 地址：** 我们不收集或存储您的原始 IP 地址。
- **连接数据：** 我们不记录会话时长或连接时间戳。

## 2. 存储在您设备上的数据

Dash VPN 完全作为一个客户端在您的设备上独立运行。

- **服务器凭据：** 您的服务器配置信息安全地存储在您设备的**钥匙串 (Keychain)** 中。我们无法访问这些信息。
- **路由规则：** 仅保存在应用程序本地。

## 3. 诊断数据（匿名）

为了提高应用的稳定性，我们需要收集**匿名**的崩溃报告（通过 Sentry），且仅在应用发生故障时收集。

- **发送的内容：** 设备型号（如 iPhone 15）、iOS 版本以及导致崩溃的技术错误代码。
- **不发送的内容：** 不包含任何个人身份信息、不包含流量数据、**不包含 IP 地址**。
- **您的控制权：** 您可以随时在应用设置中关闭此功能。

## 4. 用户支持

如果您联系我们寻求支持（例如通过电子邮件发送日志），我们将仅使用您**自愿提供**的信息来协助您解决问题。我们不会与第三方分享这些信息。
